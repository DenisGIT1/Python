# Python

Реализованы проекты:
• Разведочный анализ и отбор признаков, генетический поиск оптимальной архитектуры CNN-GRU-Dense модели для размеченных данных, предобучение ансамбля адаптивных автоэнкодеров на        неразмеченных данных и тонкая настройка ансамблевой модели для классификации временных рядов.
•	Разведочный и регрессионный анализ на основе линейной и полиномиальной регрессии.
•	Анализ данных на основе регрессионных моделей SVM, CART и ансамблевых моделей.
•	Классификация данных на основе моделей SVM, CART, ансамблевых и нейросетевых моделей.
•	Полносвязная сверточная нейронная сеть с предобучением и дообучением для классификации изображений.
